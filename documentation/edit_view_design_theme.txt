VIEW PAGES: Document-like, professional presentation
  - White/parchment backgrounds for printability
  - Clean, spacious layouts
  - Emphasis on data presentation over interaction
  - Export-friendly designs

EDIT PAGES: Form-focused, efficient data entry
  - Dark brutalist theme consistent with dashboard
  - Clear visual hierarchy for form fields
  - Inline validation and feedback
  - Save/cancel actions prominently placed

SLUG PAGES: Content-rich, readable
  - Optimized for long-form content (blog posts, articles)
  - Typography-first approach
  - Generous line-height and spacing
  - Complementary image treatment
